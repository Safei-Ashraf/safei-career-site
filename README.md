# Safei Career Site

Single-page portfolio and reference site built with Next.js, React, and TypeScript.

## What it includes

- Codex-inspired dark layout with a sticky sidebar
- Full-screen hero / overview section
- Interactive resume section
- Web-friendly impact reference section
- Downloadable `.docx` files in `public/docs`
- SEO metadata, sitemap, robots, and structured data

## Update later

Search for `TODO: replace this` in the codebase to update:

- production URL
- email
- LinkedIn
- GitHub
- phone
- WhatsApp
- Google Docs resume link

## Commands

```bash
npm install
npm run dev
```
